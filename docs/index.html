Informazioni sul libro

Questo libro copre le competenze e le tecniche necessarie per creare applicazioni web interattive, sofisticate e accessibili. Si concentra su linguaggi, strumenti e librerie lato client che i professionisti usano per costruire i siti web che utilizzi ogni giorno. Presuppone una base di programmazione (ad esempio un corso introduttivo in Java o Python) e alcuni concetti dalle technical foundations of informatics). Questi materiali sono stati sviluppati per il corso INFO 340: Client-Side Web Development presso la University of Washington Information School; tuttavia sono stati strutturati come risorsa online per chiunque voglia imparare web programming.

Questo libro è distribuito con licenza Creative Commons Attribution-NonCommercial-ShareAlike 4.0 International.

Questo libro è open source. Visita il progetto su GitHub per contribuire con miglioramenti.
